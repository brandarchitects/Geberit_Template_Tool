# Fonts

Place the Aktiv Grotesk font files in this directory:

```
AktivGrotesk-Bold.woff2
AktivGrotesk-Bold.ttf
AktivGrotesk-Regular.woff2
AktivGrotesk-Regular.ttf
AktivGrotesk-Light.woff2
AktivGrotesk-Light.ttf
```

Both `.woff2` (preferred) and `.ttf` formats are used.
- `.woff2` is loaded in the browser for preview
- Both formats are base64-embedded into the PDF by the export engine (Puppeteer)
